Projects

Below you'll find a list of projects that I have worked on during my time as an AUT Student. 

Software Article Database Application

A group of 3 people for our "Contemporary Issues in Software Engineering" paper. Students were given a page of requirements of how the app should work from the Product owner and students were expected to come up with Sprint Goals, User Stories, and Sprint Retrospective.
A single page application where users can submit an article's information to a MongoDB Database. Users can also view all submitted articles. Below are the features I was responsible for:

  • Implemented REST API and CRUD Operations for MongoDB
  • Configured MongoDB Database and Schema
  • Website Deployment via Heroku
  • CI/CD using GitHub Actions
  • Navigation using React Router

Tech Stack: Express.js, Node.js, React, REST API, CRUD Operations, MongoDB, Heroku, and VSCode.

Cab Booking System

A website where users can book a cab while an administrator assigns them a Cab manually. Bookings are submitted to an SQL Database and updated once an administrator assigns them a cab. 

  • Configured PHP server
  • Implemented all server-side logic
  • Configured MySQL Database
  • Implemented Form Validation

Tech Stack: PHP, MySQL, MyPHPAdmin, HTML and CSS.

Flux Velocity -  Racing Game

Created a small racing game with a team of 4 people. Players can race against AI, play on multiple maps, and boost their speed to win the game. 

  • Implemented Core Movement Logic
  • Implemented various UI Logic
  • Maintained core function library

Tech Stack: C#, Unity 2018 (LTS), Visual Studio 2019.
